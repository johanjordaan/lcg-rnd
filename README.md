lcg-rnd
=======

LCG (linear congruential generator) random number generator. Based on the information here: [http://en.wikipedia.org/wiki/Linear_congruential_generator].

`npm install lcg-rnd`

or

`bower install lcg-rnd`

Usage
========


```coffee
rnd = require 'lcg-rnd'
rnd.srand 10
rnd_val = rnd.random()
```
